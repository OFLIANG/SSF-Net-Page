为什么我们需要高光谱目标跟踪?

从传统RGB到高光谱,视觉追踪技术的演进之路

传统 RGB 追踪的根本局限

  • 光照敏感:光线变化导致颜色失真,目标外观剧烈改变
  • 遮挡脆弱:部分遮挡后难以重新捕获目标
  • 背景混淆:相似颜色和纹理的背景造成严重干扰
  • 伪装失效:无法区分视觉上相同但材质不同的物体

高光谱成像的独特优势

  • 数十至数百个连续波段:覆盖可见光到近红外范围
  • 丰富的光谱指纹:每种物质都有独特的光谱反射曲线
  • 揭示物理化学组成:识别材料本质而非表面颜色
  • 抗干扰能力强:阴影、弱纹理环境下仍可稳定追踪

现有方法的不足与创新契机

大多数现有工作将高光谱图像强行压缩成3通道假彩色图,然后套用传统的RGB追踪器。这种做法存在致命缺陷:

❌ 丢失了波段间的深层光谱关联信息
❌ 未能充分利用高光谱数据的维度优势
❌ 简单拼接多模态特征导致信息冗余

深入核心:三大创新模块

痛点: 传统方法将高光谱转为3通道图像送入普通的 VGG 或 ResNet,丢失了深层波段相关性。
创新: S²FB (Spatial-Spectral Feature Backbone) 专为高光谱数据打造。它采用带有残差连接的空间光谱卷积块 (S²CB),利用 2D 深度可分离卷积提取"空间纹理",同时利用 3D 卷积捕获"连续波段的光谱特征",从而获得联合的空间-光谱分层表示。

技术规格
  • • 输入维度: H×W×B (高度×宽度×波段数)
  • • 四阶段架构: Stage 1-4逐步提取多层次特征
  • • S²C层: 2D深度可分离卷积(空间) + 3D卷积(光谱)
  • • 输出分辨率: H/4 × W/4 × 8C (C=256)
关键优势
  • • 滑动3D卷积核建立多个连续光谱带之间的上下文关系
  • • 捕获物质的光谱签名
  • • 生成联合的空谱分层表示
  • • 残差连接缓解梯度消失
效果验证:消融实验表明,结合残差结构的空间与光谱双分支,比单独提取提升了近 2% 的 AUC 精度。

机制: 高光谱特征和 RGB 特征提取出来后并非各自为战。SAFM 通过全局平均池化和 1D 卷积生成通道权重。
模态内与模态间: 模块不仅进行"模态内注意力"增强自身关键特征,还利用 Softmax 计算跨 RGB 和 HS 的"模态间注意力",使网络能动态地利用清晰的 RGB 视觉提示来补充高光谱分支缺失的纹理细节。

模态内注意力

通过1D卷积和Sigmoid激活函数,为每个通道的特征分配权重,增强重要光谱带的表达。

跨模态注意力

拼接RGB和HS嵌入向量,使用Softmax生成动态注意力权重,捕捉模态间相关性。

H_fused = H_intra_refined + H_inter_refined
Cross-Modality Self-Adaptive Weighting Channel Attention

在深度学习的特征空间里,如何判断当前画面中哪一块是你要追踪的目标?
原理引入: 借鉴遥感领域的 光谱角映射 (SAM),SAAM 将特征图投影到子空间中,计算模板特征向量与搜索区域特征向量之间的 余弦相似度(夹角)。

cos θ(i,j) = <f(z)(i,j,:), f(x)(i,j,:)> / (||f(z)|| · ||f(x)||)

其中 z 和 x 分别代表HS模态的模板图像(TI)和搜索图像(SI)

  • 目标相似(正样本): 夹角小,余弦值大 → 高置信度
  • 背景干扰(负样本): 夹角大,余弦值小 → 低置信度
SAAL 损失函数

作者引入了一种三元组损失 (Triplet Loss) 思想,专门设计了 SAAL,强迫网络在训练时缩小目标与正样本特征之间的光谱夹角,扩大与负样本(背景伪装)之间的夹角。

L_SAAL = Σ [Sim(a,p) - Sim(a,n)]

p ∈ E₁ (正样本区域), n ∉ E₂ (负样本区域)

效果:这就是 SSF-Net 能在视觉相似物体的遮挡、伪装下不跟丢的"杀手锏"。

总结与展望

研究总结

本文提出了SSF-Net,一个创新的空谱融合网络,用于高光谱目标跟踪。通过三个核心模块的协同工作:

  • S²FB - 专用的空间光谱特征提取骨干网络
  • SAFM - 智能的双模态注意力融合机制
  • SAAM + SAAL - 基于光谱角的精确目标定位

SSF-Net在多个基准数据集上取得了最先进的性能,同时在计算效率方面表现出色。
