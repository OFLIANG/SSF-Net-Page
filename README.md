# SSF-Net: 空间光谱融合网络深度解析

本项目提供了一个基于 Web 的现代化交互式讲解网站，旨在深入解读论文 **"SSF-Net: Spatial-Spectral Fusion Network with Spectral Angle Awareness for Hyperspectral Object Tracking"**。

## 🚀 快速开始

1. **直接打开**：双击 [`index.html`](index.html) 或 [`光谱融合详解.html`](光谱融合详解.html) 即可在浏览器中查看。
2. **本地服务器**（推荐）：
   ```powershell
   python -m http.server 8000
   ```
   然后在浏览器访问 `http://localhost:8000`。

## ✨ 项目特色

*   **双网站联动**：两个网站之间建立了完整的双向导航链接，方便用户在学习基础和深入实现之间自由切换
*   **智能引导系统**：
    *   在 SSF-Net 详解页面提供"新手入门指南"，引导初学者先学习基础知识
    *   在光谱融合详解页面提供"进阶学习入口"，帮助已掌握基础的用户快速跳转到实现细节
*   **单文件架构**：所有 HTML、CSS (Tailwind) 和 JavaScript 逻辑整合在一个文件中，便于分享和离线使用。
*   **全平台兼容**：针对手机、平板和电脑进行了响应式优化，支持触摸交互。
*   **动态交互**：包含手风琴折叠面板、滚动渐入动画、关键词高亮 Tooltip 等现代 UI 元素。
*   **数据可视化**：集成了 Chart.js，直观展示 HOTC-2020 和 BihoT 数据集的实验对比结果。
*   **学术质感**：采用玻璃拟态设计和紫色渐变主题，兼顾专业性与美观度。
