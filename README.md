## 🛠️ 技术栈

*   **HTML5 & Tailwind CSS**: 负责页面结构与现代化样式。
*   **Vanilla JavaScript**: 处理导航、手风琴交互及滚动监听。
*   **Chart.js**: 用于绘制实验性能对比图表。
*   **FontAwesome**: 提供丰富的矢量图标支持。

## 📝 内容概览

### index.html - SSF-Net论文解析
1.  **研究背景**：分析传统 RGB 追踪的局限与高光谱成像的优势。
2.  **系统架构**：图解 SSF-Net 的双分支结构与特征融合流程。
3.  **核心模块**：详细拆解 S²FB、SAFM 和 SAAM 三大创新组件。
4.  **实验验证**：展示消融实验、属性分析及运行效率对比数据。

### 光谱融合详解.html - 技术全景讲解
1.  **基础知识**：高光谱成像原理、数据结构、与传统图像的区别。
2.  **融合原理**：为什么需要融合、数学建模、融合层次、光谱角感知。
3.  **主流方法**：传统方法（PCA、小波变换）vs 深度学习方法（CNN、Transformer）。
4.  **应用场景**：精准农业、遥感监测、目标跟踪等实际应用领域。
5.  **SSF-Net详解**：深入剖析网络架构、核心模块工作原理及实验结果。
