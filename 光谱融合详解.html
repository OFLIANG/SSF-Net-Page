前往详解

一、高光谱成像基础

1.1 什么是高光谱图像？

核心概念

高光谱图像（HSI）是三维数据立方体，每个像素包含连续的光谱信息，通常有100-300个波段。

数据结构

  • 空间维度：H × W
  • 光谱维度：B个波段
  • 数据格式：R ∈ R^(H×W×B)

1.2 不同类型图像对比

特性 RGB 多光谱 高光谱
波段数 3 4-20 100-300+
分辨率 低 中 高

1.3 处理挑战

数据量大

数百个波段导致存储和处理成本高

维度灾难

高维空间样本稀疏，需专门策略

冗余性强

相邻波段高度相关，影响效率

二、光谱融合原理

2.1 为什么需要融合？

空间和光谱信息同等重要但各有侧重：

仅用空间的局限

  • • 无法区分相近材质
  • • 对光照敏感

仅用光谱的局限

  • • 忽略形状纹理
  • • 边界定位不准

2.2 融合建模

基本框架

输入：X ∈ R^(H×W×B)

输出：F(i,j) = f(X_spatial, X_spectral)

2.3 融合层次

数据级融合

原始数据层面组合，简单但可能引入噪声

特征级融合

分别提取特征后融合，SSF-Net采用此策略

决策级融合

独立判断后综合，适用于多模态任务

2.4 光谱角感知

光谱角（SA）量化两个光谱的相似性：

SA(x1, x2) = arccos( / (||x1|| * ||x2||))

优势
  • ✓ 对光照不敏感
  • ✓ 反映材质属性
SSF-Net应用
  • • 构建SAAM注意力机制
  • • 引导特征融合

五、SSF-Net深度解析

5.1 整体架构

SSF-Net采用双分支结构，分别处理空间和光谱信息，然后通过三个核心模块进行融合：

S2FB

空间光谱融合块

SAFM

光谱角特征调制

SAAM

光谱角注意力

5.2 S2FB - 空间光谱融合块

S2FB负责初步融合空间和光谱特征，通过并行的卷积分支提取不同尺度的信息

空间分支

使用2D卷积捕捉空间上下文和几何结构

光谱分支

使用1D卷积沿光谱维度提取频谱特征

5.3 SAFM - 光谱角特征调制

SAFM利用光谱角信息动态调整特征图的通道权重，增强判别性强的光谱响应

工作流程
  1. 计算查询和模板的光谱角
  2. 基于角度生成调制系数
  3. 对特征图进行逐通道加权
  4. 强化与目标光谱相似的响应

5.4 SAAM - 光谱角注意力机制

SAAM是SSF-Net的核心创新，将光谱角作为注意力权重的计算依据

注意力计算

通过光谱角相似度构建空间注意力图和通道注意力图

  • • 空间注意力：聚焦目标区域
  • • 通道注意力：选择关键波段
优势效果
  • ✓ 抑制背景干扰
  • ✓ 增强目标显著性
  • ✓ 提高抗遮挡能力
  • ✓ 改善小目标检测
